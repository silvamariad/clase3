Comprueba el valor del enlace autor[editar]

Comprueba el valor del |enlace autor=

El |enlace autor= enlaza al correspondiente artículo del autor en Wikipedia. Debe contener únicamente el nombre del artículo ya que el módulo de Citas valida que el |enlaceautor= no se corresponda a un enlace externo de la forma http://..., https://..., etc.

Para resolver el error, asegúrate que el valor del |enlace autor= es el nombre completo de un artículo de la Wikipedia (sin corchetes) y no un enlace a un lugar externo.

Las páginas con este error se categorizan automáticamente en la categoría Wikipedia:Páginas con referencias con enlace autor incorrecto.

Errores en ISBN[editar]

|isbn= incorrecto

Véase también: ISBN

Para las plantillas de referencias que contienen |isbn= se válida que el ISBN tiene la longitud correcta (diez o trece dígitos), que el ISBN utiliza los separadores opcionales correctos (espacios o guiones) y que el último dígito de control es correcto de acuerdo a la especificación del ISBN. Solo se permite un ISBN en este campo porque el valor completo del |isbn= se incluye en los metadatos COinS. Más de un ISBN, o caracteres que no forman parte del ISBN, hacen que no funcionen correctamente los metadatos COinS así como el enlace a Wikipedia:Fuentes de libros.

Para resolver este error, asegúrate que el valor de |isbn= es correcto, que solo se utiliza un ISBN, que se utilizan los separadores opcionales correctos y que no se incluye ningún otro texto. Usa el ISBN de 13 dígitos cuando esté disponible. Si no es así no conviertas un ISBN de 10 dígitos en uno de 13 añadiendo al principio del mismo 978 porque los dígitos de control son distintos y dará un error.

Si se utiliza un ISBN de 10 dígitos, comprueba que si el dígito de control es una 'x' en minúsculas, cámbialo a mayúsculas 'X'.

En casos muy extraños, las editoriales han publicado libros con ISBNs incorrectos. Si estás seguro de que el |isbn= corresponde verdaderamente a la obra publicada, puedes agregar |ignore-isbn-error= true a la cita, con el fin de suprimir el mensaje de error. En muchos casos, los libros pueden haber sido reeditados con un ISBN corregido. Utiliza un ISBN corregido cuando sea posible.

Las páginas con este error se categorizan automáticamente en la categoría Wikipedia:Páginas con ISBN incorrectos.

Errores en DOI[editar]

|doi= incorrecto

Véase también: DOI

A veces ocurre que el parámetro |doi= tiene el prefijo 10. en el identificador de registro. Además, no termina con una puntuación ni debe contener espacios. |doi= no está perfeccionado.

Utilice un DOI corregido cuando sea posible.

Las páginas con este error se categorizan automáticamente en la categoría Wikipedia:Páginas con DOI incorrectos

Errores de URL[editar]

URL incorrecta[editar]

|url= incorrecta

Véase también: URL

Se trata de un error en el parámetro |url=, normalmente debido a la ausencia del protocolo de transferencia de hipertexto (http). Para solucionarlo, simplemente se debe anteponer http:// a la página web. Por ejemplo:
Mal: 

cita web|url=www.rae.es

Bien: 

cita web|url=http://www.rae.es

También existe un segundo caso en que la URL puede no funcionar. Al copiar y pegar una dirección directamente desde la barra de direcciones en Wikipedia, las tildes se transforman en el código %C3%AD. Esto puede hacer que luego, al cliquear el enlace respectivo, el vínculo no funcione.

Para solucionarlo, luego de pegar la dirección en la página de Wikipedia, debemos corregir el código %C3%AD y reemplazarlo por la vocal con su tilde respectivo, por ejemplo:
Mal: 

cita web|url=http://es.thefreedictionary.com/cr%C3%ADtica

Bien: 

cita web|url=http://es.thefreedictionary.com/crítica

Las páginas con este error se categorizan automáticamente en la categoría Wikipedia:Páginas con URL incorrectas.

Autorreferencias dentro de la URL[editar]

|url= incorrecta con autorreferencia

Véase también: Wikipedia:Evita las autorreferencias

Se trata de un error en el parámetro |url=, normalmente debido a que incluye un enlace a un artículo de la Wikipedia.

Las páginas con este error se categorizan automáticamente en la categoría Wikipedia:Páginas con autorreferencias.

coautores requiere autor[editar]

|coautores= requiere |autor=

|coautor= y |coautores= son parámetros obsoletos. Sin el parámetro |autor=, o |autores=, el contenido |coautor= o |coautores= no se muestra.

Para resolverlo, use |autor1=, |autor2=, etc. para listar individualmente a los autores.

Las páginas con este error se encuentran en la categoría Wikipedia:Páginas con referencias sin autor y con coautores.

Parámetros obsoletos[editar]

La referencia utiliza el parámetro obsoleto |????= (por defecto está oculto)

A veces, con el paso del tiempo, algunos parámetros están desactualizados. Si pasa eso, se considera este parámetro como obsoleto. Para corregir este error, modifica los parámetros obsoletos.

Las páginas con este error se encuentran en la categoría Wikipedia:Páginas con referencias con parámetros obsoletos.

Parámetros redundantes[editar]

|param1=, |param2=, y |param3= redundantes

Este error ocurre cuando dos o más parámetros activan la misma cosa (ejemplo: |accessdate= y |fechaacceso= son traducciones).

Para corregir este error, retira o modifica los parámetros redundantes.

Las páginas con este error se encuentran en la categoría Wikipedia:Páginas con referencias con parámetros redundantes‎

Parámetros desconocidos[editar]

En caso de no encontrar parámetros sugeridos[editar]

Parámetro desconocido |????= ignorado

Para las plantillas de referencias que contienen parámetros desconocidos, por lo general, falta de texto, otros mal traducidos o con errores de escritura. A veces, se pueden escribir en inglés ciertos parámetros (|accessdate=) pero no parcialmente (|acessdate=).

Para corregir este error, usa el parámetro correcto en reemplazo al parámetro ignorado o sugiérelo, editando o discutiendo en la página Módulo:Citas/Sugerencias.

Las páginas con este error se encuentran en la categoría Wikipedia:Páginas con referencias con parámetros desconocidos

En caso de encontrar parámetros sin nombre[editar]

Texto "????" ignorado

A diferencia de las demás plantillas usadas en Wikipedia, las plantillas del módulo:Citas no pueden usar parámetros posicionales o sin nombre. Sin embargo, si el texto entre las 2 | (barras verticales) no lleva un signo =, se ignora el texto y se muestra como un mensaje de error.

Estos errores también aparecen como parte de los títulos o URL. Si | ocurre en una URL, remplacela con %7c. Si | ocurre en otro parámetro que no sea URL, remplácela con &#124; o {{!}}.

Para corregir el problema, retira el texto extraño, agrega '=', agrega el nombre del parámetro apropiado de la plantilla que quieres completar el texto con su parámetro, o codifica | en caso de títulos y URL.

Las páginas con este error se encuentran en la categoría Wikipedia:Páginas con referencias con parámetros sin nombre.

En caso de encontrar parámetros sugeridos[editar]

Parámetro desconocido |xxxx= ignorado (se sugiere |yyyy=)

Para las plantillas de referencias que contienen parámetros en mayúsculas ya sea parcial o totalmente (|XXXX=, |Xxxx= o |xxXx=), pero contienen el parámetro correcto en minúscula (|xxxx=). A veces, se puede escribir totalmente en mayúsculas ciertos parámetros (|ISBN=) pero no parcialmente (|Isbn=).

También se han detectado parámetros con errores ortográficos (un ejemplo es |fechaaceso= en vez de escribir |fechaacceso=)

Para corregir este error, usa el parámetro sugerido en remplazo al parámetro ignorado.

Las páginas con este error se encuentran en la categoría Wikipedia:Páginas con referencias con parámetros sugeridos
